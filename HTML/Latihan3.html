Font Style

Css dapat mengatur style dari teks yang ada pada halaman web yang dibuat. Terdapat beberapa property yang dapat digunakan untuk mengatur style yaitu sebagai berikut

  • text-align: digunakan Untuk mengatur perataan text,value ang dapat digunakan yaitu left, center, right.
  • color : Digunakan untuk memberikan warna pada teks.
  • Letter-Spacing : Mengatur jarak antar karakter.
  • word=spacing : Mengatur jarak antar kata.
